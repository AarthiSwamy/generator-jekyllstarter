## Running the yo generator locally
* gem install github-pages
* Install jekyll: https://jekyllrb.com/docs/installation/
* Install brew, node, npm and yeomen.
* Clone the repository
* npm link
* yo jekyllstarter
* yp jekyllstarter:newpost
* jekyll serve
